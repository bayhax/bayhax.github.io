python面试题

基本问题

  1. 列出5个常用Python标准库
    自身库：
    os 与操作系统相关联的函数
    datetime 日期和时间处理的方法
    random 生成随机数
    math 数学库函数，对底层C函数库的访问
    zlib 通用的数据打包和压缩
    pymsql 连接mysql数据库
    threading 线程管理
    multiprocessing 进程管理
    queue 队列
    第三方库：
    django
    flask
    requests
    virtualenv
    selenium
    scrapy
    xadmin
    celery
    re
    hashlib
    md5
    常用科学计算库：
    Numpy
    Scipy
    Pandas
  2. Python内建数据类型有哪些？
    字符串(str)—不可变
    列表(list)—-可变
    元组(tupple)–不可变
    集合(set)—–可变
    字典(dict)—-可变
    数字(int)—-不可变
    布尔(bool)—不可变
